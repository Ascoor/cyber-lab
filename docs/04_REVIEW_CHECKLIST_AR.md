# قائمة المراجعة بعد كل تنفيذ

استخدم هذه القائمة قبل اعتماد أي تعديل:

- [ ] هل التعديل داخل نطاق المرحلة الحالية في Roadmap؟
- [ ] هل يوجد أي خطر قانوني أو استخدام خارج التصريح؟
- [ ] هل تم رفض `authorized=false` قبل تنفيذ أي فحص؟
- [ ] هل يوجد أي احتمال shell injection؟
- [ ] هل تم التأكد من عدم استخدام `shell=True`؟
- [ ] هل تم منع CIDR/ranges عند الحاجة؟
- [ ] هل تم منع إدخال flags أو أوامر من المستخدم؟
- [ ] هل يوجد timeout لأي أداة خارجية؟
- [ ] هل يوجد error handling واضح؟
- [ ] هل تم حفظ report عند وجود فحص؟
- [ ] هل endpoint موثق في README أو docs؟
- [ ] هل تم اختبار `/health`؟
- [ ] هل تم اختبار الحالة المسموحة؟
- [ ] هل تم اختبار الحالة المرفوضة؟
- [ ] هل تم تحديث `docs/05_CHANGELOG_AR.md`؟
- [ ] هل تم تحديث `docs/06_DECISION_LOG_AR.md` عند الحاجة؟
- [ ] هل لا توجد dependencies كبيرة أو غير مبررة؟
- [ ] هل لا توجد أسرار hardcoded؟
- [ ] هل لا توجد تعليمات هجومية أو exploit code؟
